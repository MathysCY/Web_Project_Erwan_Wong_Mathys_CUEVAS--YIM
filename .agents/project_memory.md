# Project Memory: EFREI MockUP Web Project

## 📝 Présentation du Projet

**Nom**: MockUP Web Project (EFREI Computer Science Department)
**Description**: Refonte moderne du site web du département informatique de l'EFREI Paris. Site statique haut de gamme axé sur l'UX/UI.

- **Version Actuelle**: v0.1.2 (Branding Officiel & UI Premium)
- **Structure**: Site multipage (HTML5, CSS3, JS Vanilla).

## 🛠️ Stack Technique & Architecture

- **Structure**: HTML5 Sémantique
- **Styling**: Vanilla CSS (Variables, Flexbox, Grid, Glassmorphism)
- **Animations**: CSS Animations (Scroll Reveal, Typing Effects, Floating Shapes)
- **Branding**: Logos SVG officiels (Efrei 2022), Icons SVG
- **Images**: Assets institutionnels (Campus, Labs)
- **Responsive**: Mobile-First, Breakpoints fluides
- **SEO**: Meta tags complets, hiérarchie Hn respectée
